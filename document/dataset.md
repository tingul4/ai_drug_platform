# 可使用的資料集整理

## 資料庫與用途

| 名稱 | 用途 |
| --- | --- |
| UniProt | 蛋白質與分子交互作用 |
| DrugBank | 已知藥物–標靶–作用機制（MoA） |
| ChEMBL | 小分子–蛋白活性數據 |
| PDB | 實驗測定的蛋白質、核酸等生物大分子三維結構 |
| AlphaFold DB | 由 AI 預測的高準確度蛋白質三維結構模型 |
| SKEMPI 2.0 | 蛋白質複合物突變前後的結合自由能變化數據 |
| ProThermDB | 蛋白質熱穩定性與摺疊的熱力學參數 |
| PDBbind | 蛋白質–配體複合物的結合親和力實驗數據 |
| BindingDB | 藥物類小分子與蛋白質標靶的結合親和力測量值 |
| MoDEL | 蛋白質分子動力學（MD）模擬軌跡資料庫 |
| MDDB | 分子動力學模擬數據的存儲與共享平台 |
| mdCATH | 針對 CATH 蛋白質結構域分類的分子動力學模擬集 |
| ZINC20 | 用於虛擬篩選的可商業購買化學化合物庫 |
| ClinTox | 藥物臨床試驗結果與毒性評估數據 |
| LiverTox | 藥物誘導性肝損傷（DILI）的臨床記錄與資訊 |
| IEDB (Immune Epitope Database) | 免疫反應相關的抗原表位（Epitope）實驗數據 |
